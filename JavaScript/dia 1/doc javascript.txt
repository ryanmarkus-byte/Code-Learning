JavaScript tem alguns tipos de dados primitivos importantes:

Number: Representa números inteiros ou de ponto flutuante. Não há distinção explícita entre int e float como em Python. Ex: 10, -5, 3.14.
String: Representa sequências de caracteres (texto), delimitadas por aspas simples (') ou duplas ("). Ex: 'Olá', "Mundo".
Boolean: Representa valores lógicos: true ou false.
Null: Um valor especial que representa a ausência intencional de um valor.
Undefined: Um valor atribuído a uma variável que foi declarada, mas não recebeu um valor.

Operadores:

Os operadores em JavaScript são muito semelhantes aos de Python e C:

Aritméticos: + (adição), - (subtração), * (multiplicação), / (divisão), % (módulo), ** (exponenciação), ++ (incremento), -- (decremento).
De Comparação: == (igual a), != (diferente de), > (maior que), < (menor que), >= (maior ou igual a), <= (menor ou igual a). Há também os operadores de igualdade estrita (===) e desigualdade estrita (!==) que comparam valor e tipo. Geralmente, é recomendado usar os operadores estritos para evitar comportamentos inesperados.
Lógicos: && (AND), || (OR), ! (NOT).
